# 🚀 Animated Login & Signup Page

A modern and responsive Login & Signup UI with smooth sliding animation built using HTML, CSS, and JavaScript.

---

## ✨ Features
- 🔥 Smooth toggle animation between Login & Signup
- 📱 Fully responsive design
- 🎨 Clean and minimal UI
- ⚡ Pure HTML, CSS, JavaScript (No frameworks)
